🎵 사운드 모듈

다양한 환경음과 ASMR 사운드를 즐기세요.

🌧️ rain_ambient.exe - 빗소리 ASMR
🐦 forest_birds.exe - 새소리 자연음
☕ cafe_noise.exe - 카페 소음
⌨️ mech_keys.exe - 기계식 키보드
🏢 office_ambient.exe - 사무실 환경음
📚 quiet_study.exe - 조용한 학습 환경

💡 팁

각 사운드 모듈을 클릭하면 해당 환경음으로 전환됩니다. 상황에 맞는 사운드를 선택하여 집중력을 높이세요.

⌨️ 키보드 단축키

효율적인 사용을 위한 키보드 단축키를 활용하세요.

  • Space 재생/정지
  • F 전체화면 토글
  • F11 브라우저 전체화면
  • 1-6 사운드 모듈 전환

🎮 컨트롤

  • RUN/STOP - 사운드 재생/정지
  • EXPAND/RESTORE - 사이드바 숨기기/보이기
  • VOL - 볼륨 조절 슬라이더

⚡ 빠른 사용법

스페이스바를 누르면 즉시 재생/정지할 수 있습니다. 숫자키 1-6으로 원하는 사운드 모듈을 빠르게 전환하세요.

🖥️ 터미널 인터페이스

실제 개발 환경과 같은 터미널 인터페이스를 제공합니다.

화면 구성

  • 실시간 시계 - 현재 시간과 날짜 표시
  • 명령어 시뮬레이션 - 실제 터미널 명령어들이 자동으로 실행
  • 상태 표시 - 현재 실행 중인 모듈과 PID 정보
  • 매트릭스 파티클 - 배경의 동적 파티클 효과

🎯 집중 모드

전체화면 모드(F11)를 사용하면 더욱 몰입적인 환경에서 작업할 수 있습니다.

🎧 사용 시나리오

다양한 상황에서 Loop Room을 활용해보세요.

코딩 작업

  • 기계식 키보드 사운드로 타이핑 감각 향상
  • 빗소리나 카페 소음으로 집중력 증진
  • 터미널 화면으로 개발자 감성 연출

학습 & 독서

  • 조용한 도서관 환경 (quiet_study.exe)
  • 자연음으로 편안한 분위기 조성
  • 사무실 환경음으로 업무 집중력 향상

🌟 추천 설정

작업 유형에 따라 볼륨을 30-70% 수준으로 조절하고, 전체화면 모드를 활용하면 최상의 경험을 얻을 수 있습니다.
